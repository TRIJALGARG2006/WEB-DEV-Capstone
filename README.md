# WEB-DEV-Capstone# Foodie 🍕 - Food Delivery Landing Page

**Foodie** is a modern, fully responsive landing page for a fictional food delivery service. It features a clean UI, attractive food categories, restaurant listings, and promotional sections designed to convert visitors into app users.

## 🚀 Features

- Responsive Design  
- Sticky Navigation  
- Hero Section  
- Search Bar  
- Dynamic Grids  
- Smooth Animations  
- Modern Typography (Montserrat)

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- Google Fonts  
- Icons8  
- Unsplash Images  

## 📂 Project Structure

```
/Foodie-Project
├── index.html
├── style.css
└── README.md
```

## ⚙️ How to Run

1. Add `index.html`  
2. Add `style.css`  
3. Open the HTML file in any browser
